# An-cli
## Herramienta de codigo abierto para ver animes subtitulado al español en consola.

#### Requisitos
1. Reproductor de video [mpv]("https://mpv.io/").
2. [NodeJS]("https://nodejs.org/es/").

## Instalación 
```bash
npm install -g an-cli
```

## Uso
```bash
an
```
